Product Filter & Card View (Vite + React)

A responsive product listing page built using **Vite + React**.  
This project features:
- A dynamic product card section fetched from a public API.
- A toggleable left-side filter panel.
- Adaptive layout: cards expand to full width when the filter is hidden.

---

Features

✅ Built with **Vite** for ultra-fast development  
✅ Fetches data from [FakeStore API](https://fakestoreapi.com/)  
✅ Toggle between **full-width cards** and **filter + card view**  
✅ Smooth layout transitions  
✅ Clean, modular component structure  

---

Tech Stack

- **React (Vite)**
- **CSS3 (Flexbox layout)**
- **FakeStore API** (for product data)
